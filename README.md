# WOW — детская студия в Ижевске

Лендинг детской студии **WOW** (аниматоры на детские праздники, г. Ижевск).
Дизайн вдохновлён [breedlove.xyz](https://breedlove.xyz) и адаптирован под детскую
студию: тёплая светлая палитра + розовый акцент из логотипа.

## Стек

- [Vite](https://vitejs.dev) — сборка
- [Tailwind CSS v4](https://tailwindcss.com) — стили (CSS-first конфиг с дизайн-токенами)
- [GSAP + ScrollTrigger](https://gsap.com) — анимации (интро героя, закреплённый
  скролл-сторибук «Как проходит праздник», раскрывающееся видео)

## Разработка

```bash
npm install
npm run dev      # http://127.0.0.1:5173
npm run build    # сборка в dist/
npm run preview  # предпросмотр прод-сборки
```

## Структура секций

1. **Hero** — фоновая картинка, заголовок с белой обводкой (дневной режим), бейджи, кнопки
2. **Студия** — «Создаём волшебство каждый день» + блок доверия (фичи вокруг вытянутого фото)
3. **Наши герои** — плотная монолитная сетка персонажей
4. **Как проходит праздник** — закреплённый скролл-сторибук с фото (GSAP)
5. **Цены** — редакторский прайс-лист
6. **Так это выглядит вживую** — раскрывающееся при скролле видео + плейсхолдеры галереи
7. **Отзывы** — карточки с цифрами и текстом (VK / 2ГИС)
8. **Финальный CTA** — форма слева, контакты на градиентной карточке справа

## Тема

Поддерживается дневной / ночной режим (переключатель в шапке, сохраняется в localStorage).

## Что заменить позже

- Фото персонажей в блоке «Наши герои» (`public/heroes/`)
- Фото/видео в галерее под блоком с видео (плейсхолдеры «фото скоро»)
- Видео и фото в блоке доверия при необходимости
